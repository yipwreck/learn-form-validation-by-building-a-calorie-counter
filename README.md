JavaScript Calorie Counter

This project is a simple calorie counter built using JavaScript as part of freeCodeCamp's Learn Form Validation by Building a Calorie Counter course.

Features

User input validation for calorie tracking

Dynamic form handling

Real-time error messaging

Interactive UI updates

Technologies Used

JavaScript (ES6)

HTML/CSS (for UI)

Getting Started

Clone this repository:

git clone https://github.com/yipwreck/javascript-calorie-counter.git

Open index.html in your browser.
